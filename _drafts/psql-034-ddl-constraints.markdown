Sebuah **Constraint** digunakan untuk memvalidasi / mengontrol data yang dapat di simpan pada sebuah table / kolom. **Constraint** dibagi menjadi beberapa macam yaitu:

1. Not Null
2. Check
3. Unique
4. Primary key
5. Foreign keys
