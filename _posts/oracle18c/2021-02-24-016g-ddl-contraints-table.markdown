Sebuah **Constraint** digunakan untuk memvalidasi atau mengontrol data yang dapat di simpan pada sebuah table. **Constraint** dibagi menjadi beberapa macam yaitu:

1. Not Null
2. Unique
3. Check
4. Primary key
5. Foreign keys
